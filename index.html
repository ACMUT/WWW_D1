ساختن ساختار HTML بلاگ با المنت‌های جدید HTML5

ارسال شده توسط هاشم در تاریخ چهارشنبه ۲ اسفند ۱۳۹۶

برای ایجاد این صفحه از المنت‌های معرفی شده در HTML5 که با عنوان semantic elements شناخته می‌شن استفاده شده. خوندنِ سورس کد این صفحه می‌تونه حاوی اطلاعات خوبی درباره کاربرد این المنت‌ها باشه. شاید هم نباشه :))

اطلاعات بیشتر استفاده از این تگ‌های جدید برای ساختن وبسایت رو می‌تونید در MDN جستجو کنید.

همینطور از BEM naming convention برای نام‌گذاری classها استفاده شده. این قراردادها به ماژولار نوشتن کد‌های CSS کمک می‌کنن. روش‌هایی مثل BEM، SMACSS، OOCSS و... اینجا می‌تونید درباره این روش‌ها بیشتر بخونید.
